Privacy Policy

Privacy for the current Trend2Short AI MVP is intentionally simple.

This version of the project is a static browser-based MVP designed to operate locally without a backend service.

No Personal Data Collection

The current MVP does not intentionally collect personal data from users through forms, accounts, or server-side storage.

No Backend Services

The application does not use a backend in this version, and no user generations are transmitted to a remote application server.

LocalStorage Only

Saved generations, pins, filters, and related history are stored locally in the browser using LocalStorage only.

No Data Sales

Trend2Short AI does not sell user data in the current MVP.

User Control

Users can remove saved history directly from inside the application by deleting items or using the clear history action.

Policy Scope

This privacy policy applies to the current MVP version and may change if future versions introduce accounts, analytics, backend processing, or external APIs.
